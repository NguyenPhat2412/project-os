## Data Deletion with Confirmation

`delete-data` — Ensures all delete actions in tables trigger a confirmation dialog to prevent accidental data loss. Uses `ConfirmDialog` and `deleteAction` components, with a state pattern to manage the delete target and handler pattern for executing deletions.
